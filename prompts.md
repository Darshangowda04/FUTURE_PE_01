# UGC Ad Prompts: Complete Guide

## Prompt 1: Basic UGC Ad Script Generation

### System Prompt
```
You are an expert UGC creative strategist who specializes in writing authentic, 
conversion-focused ad scripts that sound like real customer testimonials. Your ads 
feel genuine, relatable, and compelling—not corporate or salesy. You understand 
platform-specific requirements and psychological triggers that make people stop scrolling.
```

### Core UGC Script Prompt
```
Create a UGC ad script for [PRODUCT/SERVICE].

Product Details:
- Product Name: [PRODUCT_NAME]
- Category: [SKINCARE/FITNESS/SaaS/LOCAL_SERVICE]
- Price Point: [PRICE]
- Main Benefit: [TOP_BENEFIT]
- Target Customer: [CUSTOMER_ARCHETYPE]
- Pain Point It Solves: [CUSTOMER_PROBLEM]

Script Requirements:

**Hook/Opening (5-10 seconds):**
- Start with a relatable statement or surprising observation
- DO: "I thought I'd have to pay $500 for this result..."
- DON'T: "Introducing our amazing product..."
- Get viewer attention before they scroll

**Problem Setup (15-20 seconds):**
- Describe the customer's struggle authentically
- Use "I" perspective (feels personal)
- Build emotional connection

**Solution/Product Introduction (10-15 seconds):**
- Present the product naturally in context
- Show it being used (implied)
- Focus on how it fixes the problem

**Results/Benefits (15-20 seconds):**
- Share specific, believable results
- Include before/after indicators
- Use numbers when possible

**Social Proof (5-10 seconds):**
- Mention customer satisfaction, reviews, or success stories
- Keep it conversational ("everyone's obsessed with this...")

**CTA/Closing (5-10 seconds):**
- Natural, non-pushy call to action
- Urgency without desperation
- "Check it out" rather than "BUY NOW"

Tone Specifications:
- Feel: Conversational, authentic, friendly
- Speed: Natural speech pacing
- Emotion: Match target customer feelings
- Language: Simple, no jargon, speak like friends
- Vibe: Recommending to a friend, not selling

Total Runtime: 60-90 seconds
Format: Use [TIMESTAMP] for video timing markers
```

## Prompt 2: Multi-Hook Generation

### User Prompt
```
Create 5 different hooks for a [PRODUCT_TYPE] ad for [PRODUCT_NAME].

Each hook should:
- Be 5-10 words maximum
- Start with relatable statement or surprising fact
- Make viewer want to keep watching
- Appeal to [TARGET_AUDIENCE]
- Address pain point: [PAIN_POINT]

Hook Inspiration Angles:
1. **Problem Hook**: Start with the frustration/problem
   Example: "Spent $200 on skincare and STILL broken out..."
   
2. **Benefit Hook**: Lead with the result/outcome
   Example: "My skin cleared up in 2 weeks (no doctor visits)..."
   
3. **Curiosity Hook**: Create intrigue or mystery
   Example: "Nobody talks about this skincare hack..."
   
4. **Contrast Hook**: Highlight before/after contrast
   Example: "My acne routine used to cost $300/month..."
   
5. **Authority Hook**: Position as knowledgeable insider
   Example: "As a dermatologist, here's what actually works..."

For each hook, provide:
- The 5-10 word hook
- 2-3 sentence explanation of why it works
- Platform recommendation (TikTok/Instagram/YouTube)
- Estimated attention-hold rate
```

## Prompt 3: Platform-Specific Adaptation

### User Prompt
```
Adapt this UGC ad script for different platforms:

Original Script:
[INSERT_SCRIPT]

Platforms to Adapt For:
1. **Instagram Reels** (15-30 seconds)
   - Trending audio considerations
   - Text overlay suggestions
   - Hook style (visual first)

2. **TikTok** (15-60 seconds)
   - Trending sounds/audio
   - Caption generator style
   - Hook for FYP algorithm

3. **YouTube Shorts** (15-60 seconds)
   - YouTube audience expectations
   - Different hook style
   - Trust-building approach

4. **Facebook Ads** (6-15 seconds)
   - Facebook older demographic
   - Accessibility/captions needed
   - Consent angle (privacy-focused)

For each platform, provide:
- Adapted script (timing marks)
- Platform-specific hooks
- Audio/music suggestions (don't list copyrighted songs)
- Text overlay key moments
- Expected performance metrics

Keep core message consistent while optimizing for platform culture.
```

## Prompt 4: Result-Focused Script

### User Prompt
```
Write a UGC ad emphasizing specific results for [PRODUCT].

Results Focus Details:
- Main Result: [QUANTIFIABLE_RESULT]
- Timeframe: [TIME_TO_RESULTS]
- Proof Type: [TRANSFORMATION/COMFORT/CONVENIENCE/SAVINGS]
- Competitor Comparison: [COMPARISON_ALTERNATIVE]

Structure:

**Before State (Vivid Description)**
- Paint picture of struggle
- Emotional weight
- Cost/time sacrifice
- Frustration level

**Turning Point**
- Discovery of product
- Initial skepticism
- First use experience

**After State (Specific Results)**
- Exact numbers/measurements
- Emotional transformation
- Lifestyle improvement
- ROI/value delivered

**Why This Works** (Social Proof)
- How many people achieved this
- Common surprise reactions
- Long-term sustainability

**Urgency Element**
- Limited supply / special pricing
- Deadline context
- Why now matters

Requirements:
- Every benefit mentioned must be specific (not vague)
- Include measurable results (lbs, days, dollars, %)
- Feel authentic, not exaggerated
- Avoid comparison to medical claims
```

## Prompt 5: A/B Testing Variations

### User Prompt
```
Create 3 different emotional angles for the same UGC script about [PRODUCT].

Core Product Info:
- Product: [PRODUCT_NAME]
- Primary Benefit: [BENEFIT]
- Target: [AUDIENCE]

Emotional Angle 1: **Confidence/Empowerment**
- Focus: How product makes customer feel powerful/capable
- Tone: Celebratory, proud, motivated
- Hook Style: Transformation, before/after confidence shift
- CTA Style: "Join people who..." / "Become the person..."

Emotional Angle 2: **Frustration Relief/Ease**
- Focus: How product removes pain/complication
- Tone: Relieved, grateful, peaceful
- Hook Style: Relatable struggle, quick solution
- CTA Style: "Stop struggling with..." / "Finally, an answer..."

Emotional Angle 3: **FOMO/Community Belonging**
- Focus: What others are experiencing, being part of movement
- Tone: Inclusive, trendy, everyone's-in-on-it
- Hook Style: "Everyone's talking about..." / "My friends discovered..."
- CTA Style: "Don't miss out..." / "Join thousands who..."

For each angle:
- Write complete script adaptation
- Indicate target viewer psychographic
- Platform recommendation
- Expected conversion rate difference
```

## Prompt 6: Objection-Handling UGC

### User Prompt
```
Create UGC ad scripts that address common objections to [PRODUCT].

Common Customer Objections:
1. [OBJECTION_1] (e.g., "It's too expensive")
2. [OBJECTION_2] (e.g., "Does it actually work?")
3. [OBJECTION_3] (e.g., "What if I hate it?")
4. [OBJECTION_4] (e.g., "[COMPETITOR] is cheaper")

For each objection:
- Write UGC script that subtly addresses it
- Don't directly argue against objection
- Use natural testimonial format
- Prove benefit through story, not claims
- Include pricing/value discussion when relevant

Example Structure for "Too Expensive" Objection:
[HOOK about initial price skepticism]
"I was like, 'Is this worth $X?'" 
[Show results over time]
"I've been using it for 3 months and already saved $X..."
[Value articulation]
"Honestly, it paid for itself..."

Make each script feel like a real customer sharing their experience.
```

## Implementation Strategy

### Best Performing UGC Elements
✅ Imperfect lighting/video quality (feels authentic)
✅ Speaking directly to camera (builds connection)
✅ First-person storytelling (relatable)
✅ Specific numbers and timeframes (credible)
✅ Before/after visuals of product in use (proof)
✅ Natural skepticism overcome (persuasive arc)
✅ Casual language and fillers ("like", "honestly", "literally")

### Avoid in UGC Ads
❌ Corporate jargon or slick presentation
❌ Studio lighting/professional production (too polished)
❌ Generic benefit claims without specifics
❌ Obvious call-to-action sales language
❌ Green screen/artificial backgrounds
❌ Influencer-style high production value

### Recording Best Practices
- Use smartphone camera (natural quality)
- Daylight or soft indoor lighting
- Casual clothing appropriate to product
- Multiple takes of hooks (find most natural)
- Speak conversationally, not scripted
- Include product in real context
- Test sound levels for platform

## Prompt Engineering Principles for UGC

1. **Authenticity > Perfection**: Imperfect is better for UGC
2. **Emotional Journey**: Hook → Problem → Solution → Result
3. **Specificity**: "Lost 12 lbs in 8 weeks" beats "Amazing results"
4. **Relatability**: Focus on customer, not product features
5. **Platform Psychology**: Different platforms reward different pacing
6. **A/B Success Factor**: Test hooks before full production
